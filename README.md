# HR Management System

## Technologies Used
- Java Spring Boot
- React JS
- MySQL
- REST API

## Features
- Employee Management
- Payroll Management
- Leave Management

## How to Run Backend
1. Import project in Spring ToolSuite4
2. Configure MySQL database
3. Run HrmsApplication.java

## How to Run Frontend
1. Open terminal in hrms_frontend folder
2. Type: npm install
3. Type: npm start
